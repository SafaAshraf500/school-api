# school-api

School Management REST API (Node.js + Express + MongoDB)

## Structure
- app.js, server.js
- config/, controllers/, middlewares/, models/, routes/
- uploads/ (ignored)

## Setup
1. copy `.env` with your variables
2. `npm install`
3. `npm run dev` (or `node server.js`)

## Notes
- JWT auth, multer uploads, role-based auth.
